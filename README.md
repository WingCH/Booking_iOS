

# Booking_iOS

**Backend and Web version** : https://github.com/WingCH/Booking_Web

Booking_iOS is iOS client base on Booking_Web.

## Used library in GitHub

* Alamofire
* SwiftyJSON
* SwiftMoment
* CalendarKit
* PopupDialog
* Kingfisher
* EFQRCode
* SwipeCellKit
* CRRefresh
* NVActivityIndicatorView
